# Create an issue

Create a new issue within a project.

## Usage

``` r
create_issue(project, summary, description, issuetype = "Task")
```

## Arguments

- project:

  jira project name

- summary:

  summary of the issue

- description:

  more detailed description of the issue

- issuetype:

  issue type, i.e. "Task", "Bug", "Improvement", etc.
